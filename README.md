The system employs STM32 microcontrollers as terminal nodes to collect multi-sensor data via RS485 buses, utilizes LoRa technology for long-distance edge transmission to a Hi3861 gateway, and integrates a Huawei Cloud IoT platform for data management, supported by a YOLOv5 algorithm for visual hazard detection. The results demonstrate that the system achieves high reliability, with a LoRa communication success rate exceeding 93% at distances up to 1200 meters and an average end-to-end data delay of only 3.7 seconds. Furthermore, the integration of edge computing and cloud visualization enables active early warning, effectively reducing potential safety hazards. This research concludes that the proposed architecture provides a low-latency, wide-coverage, and extensible solution for the digital transformation of modern grain warehouses.
